در شصت و هشتمین جلسه فرانت چپتر، رامین رضایی (مهندس ارشد نرم‌افزار در Sinch) تجربیات خود از سه کنفرانس مطرح بین‌المللی Google Summit، Anthropic Claude Code Event و React Norway را با ما به اشتراک گذاشت و درباره مسیر جدید توسعه نرم‌افزار در عصر هوش مصنوعی گفت‌وگو کرد.

## سرفصل‌های ارائه شده

- **Prompt to Production**: مسیر تبدیل ایده و پرامپت به کد پروداکشن
- **Google Summit**: تازه‌های کنفرانس گوگل
- **Anthropic Claude Code Event**: نقش Claude در توسعه نرم‌افزار
- **React Norway**: ترندهای جدید React
- **آینده توسعه‌دهندگان**: نقش جدید توسعه‌دهندگان در تیم‌های مدرن
- **ابزارهای AI**: ابزارهایی که شیوه ساخت محصولات را تغییر می‌دهند
