در بیست و هفتمین جلسه فرانت چپتر، کامران داور به معرفی و آموزش Server Components پرداخت. Server Components رویکرد جدیدی در معماری اپلیکیشن‌های React هستند که امکان رندر کامپوننت‌ها در سمت سرور را فراهم می‌کنند.

## سرفصل‌های ارائه شده

- **معرفی Server Components**: آشنایی با مفهوم و فلسفه Server Components
- **مقایسه با Client Components**: تفاوت‌ها و موارد استفاده هر کدام
- **مزایای Server Components**: بهبود عملکرد، کاهش bundle size و SEO
- **پیاده‌سازی عملی**: نحوه استفاده از Server Components در NextJS
- **الگوهای ترکیبی**: استفاده همزمان از Server و Client Components
- **Streaming و Suspense**: رندر تدریجی با قابلیت‌های جدید React
